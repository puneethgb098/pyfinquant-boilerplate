Installation
============

PyFinQuant can be installed using pip:

.. code-block:: bash

    pip install pyfinquant

For development installation:

.. code-block:: bash

    git clone https://github.com/yourusername/pyfinquant.git
    cd pyfinquant
    pip install -e ".[dev]"

Dependencies
-----------

PyFinQuant requires:

- Python 3.8 or higher
- numpy
- pandas
- scipy
- matplotlib
- yfinance
- seaborn
- requests

Optional dependencies for development:

- pytest
- pytest-cov
- ruff
- mypy
- sphinx
- sphinx-rtd-theme
